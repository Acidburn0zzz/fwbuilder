Firewall Builder Quick Start Guide Page 0

This guide provides a brief overview of the Firewall Builder application and its key concepts.

GUI Layout

The Firewall Builder application is comprised of three primary windows shown in the screenshot:

  • Object Window. Objects in the active Library are displayed in the object tree. Objects can be dragged-and-dropped from the object tree to the Rules Window.
  • Rules Window. To open a Policy for editing, click right mouse button on the corresponding object in the tree and coose menu item "Edit". Click the '+' button at the top of the window to add a new rule.
  • Editor Window. Double-click on objects to open them for editing in the Editor Window. Changes to object attribute fields take effect immediately.
